Loading a Show

Using the BAPS Web Show Manager website you can create your entire show ready to load in BAPS Presenter.

To load a show first click on the Load Show button on the interfaace:



Enter user details

This will open the show loader. You must then enter your details to get a list of your shows:



In general you will need to select other user and enter the BAPS username that you created your show using. If you are unsure which username you should use then you can find it in the top right hand corner of the BAPS Web Show Manager interface when editing your show.
By default you will see all your shows that you set to be broadcast at any time up to 3 hours before the current time. If you need to load a show that happened further back in time that than you must de-select the checkbox saying new shows only.

Choose a show

You are next confronted with a list of shows you have created. Select one and click go:

Select the channels

The final part of loading a show will normally be set correctly and you don't need to worry about it. By default it will match each of your lists with the channel you specified for that list. If you wish to put lists on different channels or not load some lists then you must alter the selections.
All of the lists in the show you are loading appear down the left hand side and the available channels appear accross the top. You can select a maximum of one list to go in each channel although you can assign as many lists as you wish to the '-' channel which will cause that list not to load:



The final option on this form is to indicate whether you wish to clear a channel's existing list before loading the new one. By default the list will be cleared. Clearing a list DOES NOT affect any track currently playing, the playing track will remain after the rest of the list has been cleared.
