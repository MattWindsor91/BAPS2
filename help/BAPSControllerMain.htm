BAPS Controller

The BAPS Controller is a hardware device that allows switch boxes to be connected to control certain BAPS functions.
  • BAPS Paddle
    The BAPS Paddle is a small hand-held box which can be set to control a variety of functions.
  • BAPS Fader/Button start
    This will interface to the desk to enable automatic fader and button starts. This functionality is not yet complete.
  • BAPS AudioWall Keypad
    The AudioWall keypad will have a matrix of 20 buttons on it in order to control the 20 corresponding buttons on the AudioWall. This functionality is not yet complete.
