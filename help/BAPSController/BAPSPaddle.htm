BAPS Paddle

The BAPS Paddle can be used in two modes currently:

News Mode

In this mode the paddle controls playback for channel 1 with the leftmost 2 buttons, The size of the text in the large text window with the middle two buttons, and the rocker switch (right hand button) can be used to scroll up and down through the text window.

Playback Mode

In this mode each pair of buttons controls one of the three channels. The top button on each row plays whatever is currently loaded in the channel and the bottom row of buttons will either stop playback (if something is playing) or load the next item in the playlist if there is nothing playing.

Setting the mode

To set which mode the Paddle operates in open the options window by pressing "Ctrl+o" and select the mode from the dropdown list then click save. This is not a permanant solution for changing the mode of the paddle.
