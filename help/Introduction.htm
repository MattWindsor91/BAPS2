Broadcasting And Presenting Suite

BAPS is designed to make presenting easier. It aims to provide all the functionality you need to make a radio broadcast as streamlined as possible. The suite is split into many applications that work in a symbiotic relationship to give you immense power at your fingertips.

  • BAPS Server

    All of the work is done by the BAPS Server it coordinates all of the satellite applications and provides the stability of the system. BAPS Server will keep running even when systems fail around it.

  • BAPS Presenter

    The presenter interface is one of the many front-ends that can connect to the BAPS Server. It provides access to all BAPS Server features with focus on usability.

  • BAPS Controller

    The BAPS controller is a serious of interfaces to switch boxes which can be used to control certain BAPS features.

  • BAPS Web Show Manager

    The Show Manager website enables shows to be created from the comfort of your own home and upload the audio that you really want to play. In as few as 6 clicks a show can be loaded into BAPS Presenter and the show can start.

  • BAPS Music Manager

    The Music Manager helps to manage the Music Library component of BAPS. It can be used to efficiently catalogue and copy audio from any CD ready to be played on BAPS Server within 2 minutes.
